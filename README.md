    ## 🧪 示例（内部处理演示）

    **你的输入：**
    ```
    【题】力扣 20
    【码】class Solution {public: bool isValid(string s) { stack<char> st; for(char c:s){ if(c=='(')st.push(')'); else if(c=='{')st.push('}'); else if(c=='[')st.push(']'); else { if(st.empty())return false; if(c!=st.top())return false; st.pop();} } return st.empty(); }};
    【态】✅
    ```

    **大模型内部处理：**
    1. 识别：栈结构，匹配问题，时间O(n)空间O(n)
    2. 提炼：存期待值→匹配消栈→判空
    3. 找坑：空栈top()会RE，最后要判栈空防"((("
    4. 输出：

    **一句话思路**：用栈存待匹配的右括号，遇左括号压入对应右括号，遇右括号直接匹配栈顶。

    **算法步骤**：遍历字符 → 左括号压对应右括号/右括号匹配栈顶 → 栈空则合法

    **易错提醒**：
    ⚠️ 空栈时取top()：运行错误（RE），必须先判empty()
    ⚠️ 遍历结束忘判栈空：漏判左括号剩余情况（如"((("）

    **复杂度**：Time: O(n), Space: O(n)

    **代码**：
    ```cpp
    class Solution {
    public:
        bool isValid(string s) {
            stack<char> st;
            for (char c : s) {
                if (c == '(') st.push(')');
                else if (c == '{') st.push('}');
                else if (c == '[') st.push(']');
                else {
                    if (st.empty()) return false;  // ⚠️ 防空栈RE
                    if (c != st.top()) return false;
                    st.pop();
                }
            }
            return st.empty();  // ⚠️ 防左括号剩余
        }
    };
    ```
